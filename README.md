# Trang web bán máy tính
## _Project cá nhân_

## Đối tượng sử dụng
- Quản trị viên
- Khách hàng
## Chức năng từng đối tượng
### A. Quản trị viên
- Quản lý nhân viên
- Quản lý trang thông tin khách hàng
- Quản lý toàn bộ sản phẩm, nhà cung cấp : CRUD
- Quản lý hoá đơn
### B. Khách hàng
- Xem toàn bộ sản phẩm, xem chi tiết từng sản phẩm
- Tìm kiếm sản phẩm
- Thêm sản phẩm vào giỏ hàng, xem giỏ hàng
- Đăng nhập, đăng kí, quên mật khẩu, gửi mail
- Thanh toán hoá đơn
